# Pilot Cases

These three cases are set aside as an initial pilot set for testing the manual coding workflow before expanding to the full dataset.

## Selected Cases

1. Western Roman Empire
   Chosen as an imperial ancient case with long chronological coverage in the dataset.

2. Maya
   Chosen as an archaeology-heavy case to test coding from evidence that often depends on material and interpretive sources.

3. Soviet Union
   Chosen as a modern state-collapse case to contrast with ancient and pre-modern evidence.

## Why This Pilot Set

- It covers one imperial ancient case, one archaeology-heavy case, and one modern state-collapse case.
- It provides a manageable subset for testing source collection, evidence tracking, and coding consistency.
- It allows the workflow to be reviewed before full-dataset coding begins.
